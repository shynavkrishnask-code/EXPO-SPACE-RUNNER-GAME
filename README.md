# 🚀 Space Escape Runner

A sleek, fast-paced 2D space survival runner game built with **React Native** and **Expo**. Navigate your ship through a dangerous asteroid field, dodge incoming meteors, track your real-time score, and beat your local personal best!

---

## ✨ Features

- **🎮 Addictive Gameplay**: Simple controls to dodge fast-falling asteroids.
- **💾 Local High Score Storage**: Integrates `@react-native-async-storage/async-storage` to load, display, and persist your personal best score across app sessions.
- **🎨 Premium Visuals**: 
  - Dynamic, parallax space stars drifting background.
  - Flickering dual-layer thruster engine flame for the spaceship.
  - Detailed cratered meteor graphics with friction heat glows.
  - Glassmorphic HUD interfaces and a celebratory Game Over card.
- **🔄 Smart Reset**: Instant game state resets (spaceship centering, scoring, meteor positions).
- **📱 Universal Compatibility**: Fully responsive and optimized for Android, iOS, and Web build pipelines.

---

## 🛠️ Technology Stack

- **Framework**: [Expo](https://expo.dev) (v56) / React Native
- **Navigation**: Expo Router (unstable-native-tabs)
- **Local Storage**: React Native Async Storage
- **Styling & Animation**: React Native `StyleSheet`, `Animated` API, and `expo-linear-gradient`
